# SerinDisktopMate
一个基于DesktopMate和MelonLoader的免费桌宠软件，形象源于哔哩哔哩UP主 @Serin塞琳，灵感源于哔哩哔哩UP主 @伊阳_YiYang

### SerinMate（SerinDM） 便携版

#### 注意：SerinMate.exe可能引发误报，请在AV软件中添加排除项，或使用launcher.cmd



一个基于[DesktopMate](https://store.steampowered.com/app/3301060/Desktop_Mate/)和[MelonLoader](melonwiki.xyz)的免费桌宠软件，形象基于哔哩哔哩UP主 @Serin塞琳

#### 使用方法

1. 解压ZIP至任意位置
2. 运行 SerinMate.exe（如引发杀软误报并不便添加排除项请运行launcher.cmd
3. 如果桌宠出现后角色并非Serin塞琳，请使用鼠标左击下方桌宠并按F4（部分键盘需要同时按下Fn）找到  Serin塞琳桌宠.vrm  并打开

本软件尚在开发中，如有故障请提交Issue
如需在其它平台上使用请将SerinDM/profile/serin.vrm导入自己的桌宠软件，推荐Mate-Engine和Super Agent Party / Agent Party
